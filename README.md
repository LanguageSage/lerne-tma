# Lerne Telegram Mini App (TMA)

Это современное веб-приложение для изучения карточек, работающее прямо внутри Telegram. 
Оно полностью независимо от основного бота и имеет **собственную систему учета прогресса**.

## Структура
- `/api`: Бэкенд на FastAPI. Использует `lerne.db` для контента и создает `tma.db` для прогресса.
- `/app`: Фронтенд на React + Vite. Премиальный дизайн с 3D-анимациями.

## Как запустить

### 1. Запуск Бэкенда
```bash
cd tma/api
pip install -r requirements.txt
python main.py
```
Бэкенд запустится на `http://127.0.0.1:8000`.

### 2. Запуск Фронтенда
```bash
cd tma/app
npm install
npm run dev
```
Фронтенд запустится на `http://localhost:5173`.

### 3. Туннелирование (для Telegram)
Telegram требует HTTPS. Используйте **ngrok**, чтобы пробросить локальный бэкенд в интернет:
```bash
ngrok http 8000
```
Полученный HTTPS-адрес нужно использовать в настройках бота или приложении.

## Особенности
- **Glassmorphism**: Прозрачный, современный дизайн.
- **3D Flip**: Анимация переворота карточки при нажатии.
- **SM-2 SRS**: Надежный алгоритм интервальных повторений.
- **Independence**: Прогресс в TMA не пересекается с прогрессом в основном боте.

## Быстрый запуск
В этой папке есть файл **`run_tma.bat`**. Просто запустите его, и он автоматически откроет два окна: одно для Бэкенда, другое для Фронтенда.

## Как открыть с телефона (iPhone, Android) или другого ПК
Чтобы приложение было доступно по Wi-Fi:

1.  **Узнайте свой локальный IP-адрес**:
    - Откройте командную строку и введите `ipconfig`.
    - Найдите `IPv4 Address` (например, `192.168.1.50`).
2.  **Откройте ссылку на телефоне**:
    `http://192.168.1.50:5173/?user_id=ВАШ_ID`
    *(Убедитесь, что телефон подключен к той же Wi-Fi сети, что и компьютер)*.

> [!IMPORTANT]
> Мы настроили `main.py` и `vite.config.js` так, чтобы они разрешали подключения из внешней сети (`0.0.0.0`).
